About Us

About Server

HATSU SMP adalah server Minecraft dengan pengalaman survival terbaik, fitur ekonomi, roleplay, dan PvP. Mendukung Java dan Bedrock, server ini menawarkan performa stabil dan gameplay yang seimbang. Gabung sekarang dan mulai petualanganmu!

Gameplay Features

Gameplay Overview

Rasakan serunya Survival Economy! Kumpulkan money, bertualang, dan raih rank spesial. Season baru hadir setiap 2 bulan, namun pencapaian money dan star tetap berlanjut. Jangan lewatkan event mingguan dengan hadiah menarik yang siap menanti juara!
